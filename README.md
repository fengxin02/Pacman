Projekt bemutatása: Ez a projekt a BME "A programozás alapjai 3" tárgy keretében készült el. A cél egy teljesen funkcionális, grafikus felülettel rendelkező Pacman játék létrehozása volt Java nyelven.

Főbb technikai megoldások:

Grafikus felület: Java Swing könyvtár segítségével felépített UI.

Objektumorientált tervezés: Külön osztályok a játéktér, a karakterek és az irányítás kezelésére.

Játéklogika: Egyedi algoritmusok az ütközések kezelésére és a szellemek mozgásának vezérlésére.

Eseménykezelés: Billentyűzet-alapú irányítás megvalósítása.
